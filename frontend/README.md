# Vercel Frontend

Static Vite frontend for **Banerjee Diagnostic Foundation and Hospaccx**.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

- Set the project root directory to `frontend`
- Build command: `npm run build`
- Output directory: `dist`

The project uses `vercel.json` to rewrite all routes to `index.html`.
